## Overview

The **Events** module lets you create, manage, and track church events — from Sunday services and conferences to weddings, funerals, outreach programmes, and youth activities. Members can register for events, and you can track attendance and guest details.

Go to **Events** in the sidebar.

## Creating an Event

1. Go to **Events** → click **+ Create Event**
2. Fill in:

**Required:** Title, Start Date

**Optional but recommended:**
- Event Type
- Description
- Location and Address
- Start Time / End Time
- Capacity (for registration-limited events)
- Registration Required — toggle on if people need to register in advance
- Organiser — assign a staff member
- Branch — for multi-branch churches

3. Click **Save**

## Event Statuses

| Status | Meaning |
|---|---|
| **Draft** | Not yet visible/active |
| **Published** | Live and visible |
| **Cancelled** | Event has been called off |
| **Completed** | Event has taken place |

---

## Event Registrations

If **Registration Required** is enabled:

### Registering a Member
1. Open the event → click **+ Register**
2. Search for the member by name → click **Register**

### Registering a Guest
1. Open the event → click **+ Register Guest**
2. Enter their name, phone, and email → click **Register**

### Registration Statuses

| Status | Meaning |
|---|---|
| **Registered** | Confirmed to attend |
| **Attended** | Marked as present on the day |
| **Cancelled** | Cancelled their registration |

---

## Tips

- ✅ **Always set a location** — it helps members plan
- ✅ **Use Registration Required** for capacity-limited events (retreats, weddings)
- ✅ **Mark events as Completed** after they happen — keeps your upcoming list clean
- ✅ **Broadcast a reminder** the day before via Messages → Broadcasts
