## Overview

The **Profile Sharing** feature lets your church create a clean public or member-only share page for a **member** or **family**.

Instead of exposing the full internal record, Shepherd lets you choose exactly what to show, add a short introduction, include prayer points, and publish a safe share link for prayer support, fundraising context, care mobilisation, or spotlight moments.

You can use it for things like:
- sharing a member's prayer needs with the church
- spotlighting a family that needs support
- creating a clean page for follow-up, care, or outreach
- generating a QR code for events, meetings, or announcements

---

## What Gets Shared?

Only the content you choose.

A shared profile can include:
- **Title**
- **Short intro**
- **Selected profile fields only**
- **Prayer points**
- **Call to action**
- **Profile photo** *(if available and selected in the shared view)*

> Shepherd does **not** publish the full member or family record automatically. The shared page is curated.

---

## Where to Find It

You can start from either:
- **Members** → open a member record → **Share Profile**
- **Families** → open a family record → **Share Profile**

You can manage all created share pages at:
- **Communication → Shared Profiles**

## Creating a Shared Profile

1. Open a **member** or **family** record
2. Click **Share Profile**
3. Fill in the share builder:
   - **Title**
   - **Intro**
   - **Prayer points**
   - **Call to action** *(optional)*
   - **Fields to display**
4. Choose the access mode
5. Click **Save Draft** or **Publish**

If you save as draft, the page is not yet visible to viewers.

---

## Access Modes

When publishing, choose how the page should be accessed:

| Access Mode | What It Means |
|---|---|
| **Public** | Anyone with the link can open it |
| **Unlisted** | Not shown inside normal app navigation, but anyone with the link can view it |
| **Members Only** | Only signed-in church members can open it |

> Use **Members Only** for more sensitive prayer or care-related sharing.

---

## Publishing a Shared Profile

After you click **Publish**, Shepherd generates:
- a **share link**
- a **QR code**

You can copy the link and send it through:
- WhatsApp
- SMS
- Email
- church announcements
- printed materials using the QR code

## What Viewers See

The viewer sees a clean share page with only the approved content.

Depending on what you included, the page may show:
- name or family name
- selected profile details
- intro text
- prayer points
- action prompt

This page is separate from the full Shepherd dashboard and is designed for safe sharing.

---

## Managing Shared Profiles

Go to **Communication → Shared Profiles** to manage all created share pages.

From there, authorized users can:
- view all drafts and published shares
- see profile status
- copy the live link
- duplicate a shared profile
- revoke a published profile
- review audit history

Typical statuses include:
- **Draft**
- **Published**
- **Revoked**
- **Expired**

---

## Revoking a Shared Profile

If you no longer want a page to be accessible:

1. Go to **Communication → Shared Profiles**
2. Find the published profile
3. Click **Revoke**
4. Confirm

Once revoked, the old link will no longer open the active shared page.

## Best Practices

- ✅ Share only the fields that are truly needed
- ✅ Use **Members Only** for private or sensitive cases
- ✅ Keep prayer points clear and concise
- ✅ Review the preview before publishing
- ✅ Revoke links when the situation has passed
- ✅ Prefer family sharing when the need affects the whole household

---

## Suggested Church Uses

- **Prayer spotlight** during services or midweek meetings
- **Care support** for bereaved or struggling families
- **Fundraising context** where the church wants members to understand a need
- **Welcome spotlight** for introducing a new pastoral family or ministry leader
- **Follow-up coordination** with a simple QR code on-screen

---

## Permissions

Access depends on the permissions assigned to the user account.

Typical permissions used are:
- **Profile Share View**
- **Profile Share Create**
- **Profile Share Edit**
- **Profile Share Publish**
- **Profile Share Revoke**

If someone can view shared profiles but cannot publish or revoke them, they likely have limited access.

---

## Tips

- If a page should be visible outside the church, use **Public** or **Unlisted**
- If the page contains sensitive care information, use **Members Only**
- For live events, display the **QR code** on screen so members can open the page quickly
- If you need a fresh version of an older profile, **duplicate** it instead of rebuilding from scratch
